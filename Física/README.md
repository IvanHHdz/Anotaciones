# Física

Este directorio está orientado a anotaciones, explicaciones y ejemplos de física. La gran mayoría de estos apuntes han sido guardados como notebooks de Pluto (Julia). Así que para visualizarlos es necesario hacer uso de tanto `Pluto.jl` como de `julia`.

Para más información, consulte el [directorio de Julia](../Julia/README.md) de este repositorio o busque información sobre Pluto en internet. Si algo falla, busque información en el directorio de Julia de este repositorio, encontrará un documento de [problemas comunes](../Julia/PROBLEMAS_COMUNES.md).

[Este directorio](./Electromagnetismo/README.md) es para la rama de electromagnetismo. Principalmente correspondiente a las clases de la UNAH de _Electricidad y Magnetismo I (FS-321)_ y _Electricidad y Magnetismo II (FS-415)_.
