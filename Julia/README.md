# Julia

Este directorio recopila algunas de las prácticas y ejemplos en Julia que no pudieron colocarse en otro directorio. Además de contar con el archivo de [problemas comunes](./PROBLEMAS_COMUNES.md), que busca ayudar con los problemas comunes al tratar de usar julia y pluto.

Recuerde que puede descargar Julia desde la [página oficial](https://julialang.org/downloads/). Además de instalar Pluto con la [guía oficial](https://plutojl.org/#install).
